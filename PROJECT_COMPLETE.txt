💡 PRO TIPS FOR SUCCESS

  1. Test locally first
     → Open index.html in browser
     → Try all 15 effects
     → Verify everything works

  2. Create the workflows folder
     → Important! Don't forget this
     → Move deploy.yml inside
     → Or GitHub Actions won't run

  3. Use "GitHub Actions" source
     → Settings → Pages
     → Not "Deploy from branch"
     → Use "GitHub Actions"

  4. Wait for deployment
     → First deploy takes 2-3 minutes
     → Each update takes 2-3 minutes
     → Watch the Actions tab

  5. Hard refresh after updates
     → Ctrl+Shift+R (Windows)
     → Cmd+Shift+R (Mac)
     → See latest version

🚨 TROUBLESHOOTING QUICK GUIDE

  Site won't load?
  → Check GitHub Pages enabled
  → Wait 3+ minutes
  → Hard refresh browser

  Old version showing?
  → Hard refresh (Ctrl+Shift+R)
  → Wait 3+ minutes
  → Clear browser cache

  Effects broken?
  → Check browser console (F12)
  → Check JavaScript is enabled
  → Try different browser

  Copy/Download not working?
  → Try Chrome or Firefox
  → Check download settings
  → Works online only

  GitHub Actions failed?
  → Check Actions tab for error
  → Verify push to main branch
  → Check .github/workflows/ folder exists

  Dark mode not saving?
  → Check if localStorage enabled
  → Not in private/incognito mode
  → Clear site data and reload
