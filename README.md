# ✨ Advanced Text Effects Generator

A modern, feature-rich text effects generator built with vanilla HTML/CSS/JavaScript. No frameworks, no dependencies—just pure web magic that works everywhere!

**Live Demo:** https://abm-um5.github.io/text-effects/

## 🚀 Features

### 15+ Text Effects
- 🔥 **Bold Unicode** - Mathematical bold characters
- ⚡ **Glitch Text** - Chaotic overlapping characters
- 🙃 **Upside Down** - Flip text vertically
- 🔤 **Small Caps** - Tiny capital letters
- ✍️ **Italic Unicode** - Slanted text
- 🚫 **Strikethrough** - Line through text
- 👻 **Zalgo Text** - Creepy overlapping text
- ⬆️ **Superscript** - Raised tiny text
- ⬇️ **Subscript** - Lowered tiny text
- ‖ **Double Strike** - Double line through
- ⭕ **Circled** - Text in circles
- ◼️ **Squared** - Text in squares
- 🤡 **Mocking Case** - AlTeRnAtInG cAsE
- 🔄 **Reversed** - Backwards text
- 🫧 **Bubble Text** - Text in bubbles

### Modern UI Features
- ✨ **Live Preview** - See changes in real-time
- 🎨 **Dark Mode** - Auto-detect system preference
- 🔍 **Effect Search** - Find effects by name
- 🔗 **Combine Effects** - Stack multiple effects
- 📋 **Copy to Clipboard** - One-click copying
- 📥 **Download as Image** - Save text as PNG
- ⭐ **Favorites** - Save your best creations
- 📜 **History** - Track recent transformations
- 📱 **Mobile Responsive** - Works on all devices
- 💾 **Local Storage** - Persists data offline

## 🎯 Quick Start

### Option 1: Use the Live Site
Visit: https://abm-um5.github.io/text-effects/

### Option 2: Fork & Deploy Your Own

1. **Fork the repository** on GitHub
2. **Enable GitHub Pages**
   - Go to your repo → Settings → Pages
   - Set source to "GitHub Actions"
   - Wait ~5 minutes for deployment
3. **Your site is live!** At `https://yourusername.github.io/text-effects/`

### Option 3: Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/abm-um5/text-effects.git
   cd text-effects
   ```

2. **Open in browser**
   - Double-click `index.html` or
   - Use any local server (Python, VS Code Live Server, etc.)

## 📝 How to Use

1. **Enter text** in the textarea
2. **Click effect buttons** to apply transformations
3. **Combine effects** by clicking multiple buttons
4. **Copy result** with the copy button
5. **Download as image** to save your creation
6. **Add to favorites** to save your best work

### Pro Tips
- Click multiple effect buttons to combine effects
- Search effects by name using the search bar
- Enable Live Preview for real-time updates
- Your history and favorites are saved locally
- Works completely offline!

## 🤖 Auto-Deployment

This project uses GitHub Actions for automatic deployment. Changes pushed to `main` or `master` branches are automatically deployed to GitHub Pages.

**GitHub Actions Workflow:** `.github/workflows/deploy.yml`

To set up for your fork:
1. The workflow file is already included
2. Enable GitHub Actions in your fork
3. Push changes to auto-deploy
